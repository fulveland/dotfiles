## Installation

### Using Git and the bootstrap script

```bash
mkdir ~/dotfiles && cd $_
git clone https://github.com/ivanreese/dotfiles.git && cd dotfiles && source bootstrap.sh
```

To update, `cd` into `dotfiles` and then:

```bash
source bootstrap.sh
```
